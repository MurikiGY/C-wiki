# Importação de blibliotecas:
Para importar uma biblioteca em c, basta inserir no inicio do arquivo.
EX:
- #include <stdio.h>
- #include <math.h>
- #include <string.h>

OBS: Algumas bibliotecas como math.h necessitam de flags de compilação
